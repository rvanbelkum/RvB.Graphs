# RvB.Graphs
Contains:
- `Graph<T>`: generic directed or undirected graph. Implemented as an adjacency list.
- `Dijkstra` and `AStar` shortest path algorithms
- `BellmandFord`
- `Bresenham`: line, circle and ellipse 'drawing' in a dicrete plane.
- `FloydWarshall`
- Support to generate GraphViz .dot file for visiualization purposes.
